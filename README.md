### currency converter

A **live currency converter** built with **HTML**, **CSS**, and **Java Script**.

-------
**Live Demo:**  https://sadikashik2.github.io/Currency-Converter/

### Features

1. **Responsive Design**
   - Works on **desktop, tablet, and mobile**.

2. **Dynamic Dropdowns**
   - Select currencies from a list of countries.
   - Flags update automatically based on selected currency.

3. **Exchange Rate Calculation**
   - Fetches real-time exchange rates from a free API.
   - Convert any amount between currencies.

4. **Swap Functionality**
   - Swap From and To currencies with a single click.
   - Flags update automatically when swapped.

5. **Input Validation**

-------

### Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Free Currency API**
